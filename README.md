# Localized Apple TV Fusion Folders

A collection of localized Fusion genre folders inspired by AppleTV design.

The folders are configured to filter movies and tv shows on genre, the're sorted by popularity and localized.



## Generator

The resources in this gallery were created using a dedicated generator included in this project.

For more information on how the generator works or to use it, please refer to the specific documentation in the [Generator README](generator/README.md).
